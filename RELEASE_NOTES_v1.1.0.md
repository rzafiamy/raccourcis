# Raccourcis v1.1.0 — Advanced Debugging & Local AI Integration

This release focuses on making the application robust for power users and developers using local AI inference servers like **Makix-LLM**.

## 🚀 New Features
- **Dual-Channel Debugging**: Logs are now mirrored to both the browser console (with rich formatting) and the system terminal.
- **Request/Response Inspection**: A new `Debug Mode` in settings allows you to view full JSON payloads for AI, Vision, TTS, ASR, and Image Generation requests.
- **Native File Picker**: The "Describe Image" shortcut now uses a native system file picker instead of a manual path input.

## 🧠 Local AI & Vision Optimizations
- **Client-Side Image Resizing**: Automatic high-performance resizing of images (max 1024px) before transmission to prevent memory errors (500 OOM) on local inference servers.
- **Multimodal Streamlining**: Refactored Vision payloads to use a single-role 'user' structure, ensuring maximum compatibility with local LLaVA-style models and "text completion" workflows.
- **Model Synchronization**: Vision tasks now automatically follow your primary AI model settings by default.

## 🛠️ Internal Improvements
- **Standardized Endpoints**: Restored full compatibility with OpenAI-standard providers while keeping Makix-specific optimizations.
- **Improved Reliability**: Added binary data handling and fail-safe fallbacks for image processing in the renderer.

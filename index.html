The Hierarchy of Automation

Shortcut

The top-level container. Your named workflow, ready to be triggered manually or via cron.

Actions

Individual tasks like reading clipboard or calling AI.

Steps

The sequence of actions. Data flows from top to bottom.

Output

The final result (Text, Image, etc.).

AI Provider

Works with any OpenAI-compatible endpoint (OpenAI, Mistral, Ollama, LM Studio…)

Firecrawl

Web scraping & crawling API. Get your key at firecrawl.dev.

Google Search & YouTube

One Google Cloud API key can serve both Custom Search and YouTube. Enable the relevant APIs in your Google Cloud Console.

YouTube Data API

Uses the same Google API key as Google Search. Enable YouTube Data API v3 in Google Cloud Console.

OpenWeatherMap

Free API key available at openweathermap.org — the free tier covers current weather.

Wikipedia

Wikipedia uses a public REST API — no API key required. This tab is for reference only.

The Wikipedia action works out of the box with no configuration needed.
Google Calendar

Requires an OAuth 2.0 Bearer token with the calendar.readonly scope. Generate one via the Google OAuth Playground.

Gmail

Requires an OAuth 2.0 Bearer token with the gmail.send scope. Generate one via the Google OAuth Playground.

GitLab

Works with GitLab.com or any self-hosted GitLab instance. Create a Personal Access Token with read_api scope.

Nextcloud

Connect to your Nextcloud instance. Use an App Password instead of your main password.

Supabase

Relational database and auth. Manage your tables via PostgREST.

SMTP (Custom Mail Server)

Send emails via any SMTP server. Gmail users should use an App Password.

QR Code Generator

QR codes are generated entirely client-side using the free api.qrserver.com service — no API key needed.

The QR Code action works out of the box. No configuration required.
